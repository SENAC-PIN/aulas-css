Exercício 025: CSS Overflow

Objetivo: Controlar o comportamento de um conteúdo maior que sua caixa.

Proposta: Teste visible, hidden, scroll, auto, overflow-x e overflow-y.

Código base de referência

Observe como o HTML e as propriedades do tema são usados. Depois crie sua própria solução na área do estudante.

<div class="exemplo">
  <div class="caixa-rolagem">Conteudo grande dentro de uma caixa pequena. Role para ler tudo que aparece aqui.</div>
</div>

<style>
/* GUIA DA ATIVIDADE: Teste visible, hidden, scroll, auto, overflow-x e overflow-y. Comece pela declaração overflow indicada abaixo. Salve e compare o antes e o depois. */
.caixa-rolagem {
  width: 220px;
  height: 90px;
  overflow: auto; /* altere esta declaração e observe o efeito */
  border: 2px solid #355070;
  padding: 12px;
}
</style>

O que fazer

  1. Crie uma caixa com largura ou altura limitada.
  2. Coloque dentro dela um texto longo, uma lista grande ou outro conteúdo que ultrapasse o espaço disponível.
  3. Aplique overflow, overflow-x ou overflow-y para controlar a rolagem ou o corte do conteúdo.
  4. Compare o resultado com visible, hidden, scroll ou auto.

Resultado esperado

A página deve mostrar uma caixa onde o conteúdo excedente é controlado de forma intencional.

Checklist antes de entregar

  • A caixa tem tamanho limitado.
  • O conteúdo passa do tamanho da caixa.
  • A propriedade overflow foi aplicada e é perceptível.
