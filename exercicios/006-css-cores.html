Exercício 006: Cores CSS

Objetivo: Praticar formas diferentes de escrever cores em CSS: HEX, RGB e HSL.

Proposta: Troque uma cor HEX por RGB e outra por HSL.

O que fazer

  1. Crie três blocos, etiquetas ou cartões dentro da área do estudante.
  2. Aplique uma cor usando HEX, uma usando rgb() e outra usando hsl().
  3. Coloque um texto dentro de cada bloco identificando o formato de cor usado.
  4. Garanta contraste suficiente para que o texto fique legível.

Resultado esperado

A página deve exibir três elementos coloridos, cada um usando uma sintaxe de cor diferente.

Checklist antes de entregar

  • Há uma cor em HEX.
  • Há uma cor em rgb().
  • Há uma cor em hsl().
  • Os textos continuam fáceis de ler.
