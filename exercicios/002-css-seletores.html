Exercício 002: Seletores CSS

Objetivo: Treinar a escolha correta dos elementos que receberão estilos.

Proposta: Altere o exemplo para testar outro valor ligado a Seletores CSS.

O que fazer

  1. Crie uma pequena estrutura HTML com títulos, parágrafos, links ou itens de lista.
  2. Escreva seletores específicos para alterar apenas alguns elementos, não todos.
  3. Use o seletor estudado na aula, como classe, descendente, combinador, pseudoclasse ou pseudoelemento.
  4. Teste se cada regra afeta somente o elemento esperado.

Resultado esperado

A página deve deixar evidente que elementos diferentes receberam estilos diferentes por causa dos seletores usados.

Checklist antes de entregar

  • Usou o seletor pedido pelo tema.
  • Evitou aplicar o mesmo estilo em tudo.
  • O efeito visual confirma que o seletor funcionou.
