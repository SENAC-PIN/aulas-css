Exercício 022: Posicionamento CSS

Objetivo: Controlar a posição de elementos na página.

Proposta: Teste top, right, bottom, left e inverta o z-index.

O que fazer

  1. Crie pelo menos duas caixas ou elementos visuais.
  2. Aplique position com valores como relative, absolute, fixed ou sticky.
  3. Use top, right, bottom ou left para deslocar um elemento.
  4. Se houver sobreposição, use z-index para definir qual elemento fica na frente.

Resultado esperado

A página deve mostrar um elemento deslocado ou sobreposto de forma controlada.

Checklist antes de entregar

  • Usou a propriedade position.
  • Usou ao menos um deslocamento.
  • O posicionamento final é intencional e visível.
