Exercício 075: CSS Media Queries

Objetivo: Adaptar o layout para diferentes tamanhos de tela.

Proposta: Altere o breakpoint para 800px e teste a largura da tela.

Código base de referência

Observe como o HTML e as propriedades do tema são usados. Depois crie sua própria solução na área do estudante.

<div class="exemplo">
  <div class="caixa-media">
<span class="status-desktop">Tela Grande (Desktop)</span>
<span class="status-mobile">Tela Pequena (Mobile)</span>
</div>
</div>

<style>
/* GUIA DA ATIVIDADE: Altere o breakpoint para 800px e teste a largura da tela. Comece pela declaração background indicada abaixo. Salve e compare o antes e o depois. */
.caixa-media {
  padding: 24px;
  background: #a8dadc; /* altere esta declaração e observe o efeito */
  border-radius: 8px;
  text-align: center;
  font-weight: bold;
}
.caixa-media .status-mobile { display: none; }
.caixa-media .status-desktop { display: inline; color: #1d3557; }
@media (max-width: 600px) {
  .caixa-media {
    background: #e63946;
  }
  .caixa-media .status-desktop { display: none; }
  .caixa-media .status-mobile { display: inline; color: white; }
}
</style>

O que fazer

  1. Crie uma seção com conteúdo que funcione em tela grande e pequena.
  2. Use medidas flexíveis, como %, max-width, fr ou rem.
  3. Adicione uma media query para alterar layout, tamanho ou espaçamento em telas menores.
  4. Teste diminuindo a largura do navegador.

Resultado esperado

A página deve continuar organizada em telas pequenas, sem conteúdo espremido ou cortado.

Checklist antes de entregar

  • Usou @media ou técnica responsiva equivalente.
  • O layout muda em tela menor.
  • Não há rolagem horizontal desnecessária.
