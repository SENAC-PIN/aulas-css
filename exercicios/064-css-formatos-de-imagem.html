Exercício 064: CSS Image Shapes

Objetivo: Praticar apresentação e ajuste visual de imagens com CSS.

Proposta: Teste object-fit, object-position, filter, border-radius e clip-path.

O que fazer

  1. Insira uma imagem na área do estudante usando a tag <img> ou um recurso local da pasta assets.
  2. Aplique estilos como width, height, object-fit, object-position, border-radius, filter ou clip-path.
  3. Inclua texto alternativo no atributo alt quando usar <img>.
  4. Observe se a imagem mantém boa proporção e não deforma.

Resultado esperado

A página deve mostrar uma imagem estilizada, com tamanho, corte ou efeito visual controlado por CSS.

Checklist antes de entregar

  • Há uma imagem no HTML ou no CSS.
  • A imagem foi estilizada por CSS.
  • A imagem não ficou deformada sem intenção.
