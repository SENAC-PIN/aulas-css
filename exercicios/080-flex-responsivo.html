Exercício 080: Flex Responsive

Objetivo: Organizar elementos em linha ou coluna usando Flexbox.

Proposta: Teste flex-direction, justify-content, align-items, gap e flex-wrap.

O que fazer

  1. Crie um container com pelo menos três itens dentro.
  2. Aplique display: flex no container.
  3. Use propriedades como gap, justify-content, align-items, flex-wrap ou flex-direction.
  4. Teste a largura da janela para observar como os itens se comportam.

Resultado esperado

Os itens devem ficar alinhados e distribuídos pelo container com controle de espaçamento e direção.

Checklist antes de entregar

  • O container usa display: flex.
  • Há três ou mais itens.
  • Foi usada ao menos uma propriedade de alinhamento ou distribuição.
