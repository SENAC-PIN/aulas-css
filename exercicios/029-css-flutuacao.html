Exercício 029: Flutuação CSS

Objetivo: Entender como o float faz um elemento flutuar ao lado de outro conteúdo.

Proposta: Troque float para right e ajuste as margens.

O que fazer

  1. Crie uma imagem, caixa ou destaque pequeno antes de um parágrafo.
  2. Aplique float: left ou float: right nesse elemento.
  3. Adicione texto suficiente para contornar o elemento flutuante.
  4. Use clear em um elemento posterior para encerrar o efeito quando necessário.

Resultado esperado

O texto deve contornar o elemento flutuante, e a área seguinte deve voltar ao fluxo normal quando clear for usado.

Checklist antes de entregar

  • Usou float.
  • Há texto contornando o elemento.
  • Usou clear se o próximo conteúdo precisar ficar abaixo.
