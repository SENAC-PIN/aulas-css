Exercício 095: RWD Templates

Objetivo: Compreender como RWD Templates mantém o conteúdo utilizável em diferentes tamanhos de tela.

Proposta: Implemente um template mobile-first com áreas nomeadas que mudem de posição em telas maiores.

Código base de referência

Observe como o HTML e as propriedades do tema são usados. Depois crie sua própria solução na área do estudante.

<div class="exemplo">
  <div class="template-rwd"><header>Logo e menu</header><aside>Categorias</aside><section class="conteudo-template">Conteúdo principal</section><footer>Rodapé</footer></div>
</div>

<style>
/* GUIA DA ATIVIDADE: Implemente um template mobile-first com áreas nomeadas que mudem de posição em telas maiores. Comece pela declaração grid-template-areas indicada abaixo. Salve e compare o antes e o depois. */
.template-rwd { display: grid; /* altere esta declaração e observe o efeito */ grid-template-areas: "topo" "lateral" "conteudo" "rodape"; gap: 10px; }
.template-rwd > * { padding: 20px; background: #eaf4fb; border: 1px solid #355070; }
.template-rwd header { grid-area: topo; }
.template-rwd aside { grid-area: lateral; }
.template-rwd .conteudo-template { grid-area: conteudo; }
.template-rwd footer { grid-area: rodape; }
@media (min-width: 720px) {
  .template-rwd { grid-template-columns: 220px 1fr; grid-template-areas: "topo topo" "lateral conteudo" "rodape rodape"; }
}
</style>

O que fazer

  1. Implemente um template mobile-first com áreas nomeadas que mudem de posição em telas maiores.
  2. Crie o HTML necessário dentro da área do estudante.
  3. Aplique as propriedades específicas do tema dentro da área CSS.
  4. Teste o resultado e explique na reflexão por que essas propriedades resolvem a tarefa.

Resultado esperado

O template deve ter topo, navegação lateral, conteúdo e rodapé, empilhados no celular e reorganizados no desktop.

Checklist antes de entregar

  • Usou áreas de Grid ou estrutura equivalente.
  • Incluiu @media.
  • A disposição muda entre telas pequenas e grandes.
