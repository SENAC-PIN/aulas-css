Exercício 057: Transições CSS

Objetivo: Criar uma mudança visual controlada com CSS.

Proposta: Altere o exemplo para testar outro valor ligado a Transições CSS.

O que fazer

  1. Crie uma caixa, botão ou cartão para receber o efeito.
  2. Aplique transform, transition ou animation de acordo com o tema da aula.
  3. Use :hover quando quiser que o efeito aconteça ao passar o mouse.
  4. Mantenha o movimento simples para que o resultado seja fácil de observar.

Resultado esperado

O elemento deve mudar de posição, escala, rotação, cor ou estado com uma transição ou animação perceptível.

Checklist antes de entregar

  • O efeito está ligado ao tema.
  • A mudança visual é perceptível.
  • O movimento não atrapalha a leitura.
