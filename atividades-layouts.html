Como usar a trilha

Faça os projetos na ordem dos números. Cada card tem um exemplo visual, requisitos e um arquivo próprio para o estudante editar.

Landing page responsiva

Topo Hero + chamada Rodapé
  • Criar cabeçalho com navegação.
  • Montar área principal com texto e chamada para ação.
  • Usar Flexbox ou Grid.
  • Adicionar media query para celular.
.hero {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 24px;
}
