Aula prática: CSS Overflow

O overflow controla o que acontece quando o conteúdo é maior que a caixa.

1. Comparando os valores principais

Todas as caixas abaixo têm o mesmo tamanho. O que muda é apenas o valor de overflow.

overflow: visible

O conteúdo passa para fora da caixa. Esse é o comportamento padrão. O texto continua aparecendo mesmo sem caber.

overflow: hidden

O conteúdo que não couber fica escondido. Não aparece barra de rolagem. Parte do texto será cortada.

overflow: scroll

A caixa sempre mostra barras de rolagem, mesmo quando talvez não fossem necessárias. Aqui você pode rolar para ver o resto.

overflow: auto

O navegador decide quando precisa mostrar barra de rolagem. Na prática, esse costuma ser o valor mais usado.

2. Overflow apenas na horizontal

Também podemos controlar os eixos separadamente com overflow-x e overflow-y.

Esta linha é maior que a caixa, então a rolagem aparece apenas na horizontal.

Resumo

Use overflow: auto quando quiser rolagem somente se for necessário. Use overflow: hidden quando quiser cortar o excesso de conteúdo.
