CSS Counters

Aprenda o papel de CSS Counters e pratique alterando o exemplo.

Exemplo

  1. Planejar
  2. Codar
  3. Revisar

Código base

<div class="exemplo">
  <ol class="etapas"><li>Planejar</li><li>Codar</li><li>Revisar</li></ol>
</div>

<style>
/* GUIA DA ATIVIDADE: Mude o texto exibido antes de cada item. Comece pela declaração counter-increment indicada abaixo. Salve e compare o antes e o depois. */
.etapas { counter-reset: etapa; list-style: none; padding-left: 0; }
.etapas li::before {
  counter-increment: etapa; /* altere esta declaração e observe o efeito */
  content: "Etapa " counter(etapa) ": ";
  font-weight: bold;
  color: #355070;
}
</style>
