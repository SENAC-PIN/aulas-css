Exemplo

Verifique a cor ativa (ID tem maior prioridade)

Código base

<div class="exemplo">
  <div id="elemento-unico" class="classe-especifica">
Verifique a cor ativa (ID tem maior prioridade)
</div>
</div>

<style>
div {
  padding: 20px;
  background: #f1faee;
  color: #1d3557;
}
.classe-especifica {
  color: #457b9d;
  font-weight: bold;
}
#elemento-unico {
  color: #e63946;
  border: 2px solid #e63946;
}
</style>

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
