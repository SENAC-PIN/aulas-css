CSS Opacity

Aprenda o papel de CSS Opacity e pratique alterando o exemplo.

Exemplo

Opacidade 0.5 (texto também fica opaco)
Fundo RGBA (texto permanece nítido)

Código base

<div class="exemplo">
  <div class="conteiner-opacidade">
<div class="caixa-opaca">Opacidade 0.5 (texto também fica opaco)</div>
<div class="caixa-rgba">Fundo RGBA (texto permanece nítido)</div>
</div>
</div>

<style>
/* GUIA DA ATIVIDADE: Altere opacity entre 0.2 e 1. Comece pela declaração opacity indicada abaixo. Salve e compare o antes e o depois. */
.conteiner-opacidade {
  display: flex;
  gap: 15px;
}
.caixa-opaca, .caixa-rgba {
  flex: 1;
  padding: 20px;
  color: white;
  font-weight: bold;
  text-align: center;
  border-radius: 6px;
}
.caixa-opaca {
  background: #355070;
  opacity: 0.5; /* altere esta declaração e observe o efeito */
}
.caixa-rgba {
  background: rgba(53, 80, 112, 0.5);
  color: #355070;
}
</style>

Atividade prática

Altere opacity entre 0.2 e 1.

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
