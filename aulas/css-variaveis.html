Exemplo

Tema Definido por Variáveis

Código base

<div class="exemplo">
  <div class="tema-personalizado">
    <h3>Tema Definido por Variáveis</h3>
    <button class="botao-tema">Clique Aqui</button>
  </div>
</div>

<style>
:root {
  --cor-primaria: #355070;
  --cor-secundaria: #b56576;
  --espacamento-padrao: 18px;
  --borda-raio: 8px;
}
.tema-personalizado {
  padding: var(--espacamento-padrao);
  border: 2px solid var(--cor-primaria);
  border-radius: var(--borda-raio);
}
.botao-tema {
  background: var(--cor-primaria);
  color: white;
  border: none;
  padding: 8px 16px;
  border-radius: var(--borda-raio);
  cursor: pointer;
}
.botao-tema:hover {
  background: var(--cor-secundaria);
}
</style>
