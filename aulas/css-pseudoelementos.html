CSS Pseudo-elements

Treine formas de escolher elementos específicos da página para aplicar estilos.

Exemplo

CSS cria detalhes sem mudar o HTML.

Código base

<div class="exemplo">
  <p class="frase">CSS cria detalhes sem mudar o HTML.</p>
</div>

<style>
/* GUIA DA ATIVIDADE: Use ::after para adicionar um detalhe no final. Localize no exemplo a parte relacionada ao tema e altere um valor por vez. Salve e compare o antes e o depois. */
.frase::before { content: ">> "; color: #b56576; }
.frase::first-letter { font-size: 36px; color: #355070; }
</style>

Atividade prática

Use ::after para adicionar um detalhe no final.

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
