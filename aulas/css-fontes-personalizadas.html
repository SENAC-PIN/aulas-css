CSS Custom Fonts

Aprenda o papel de CSS Custom Fonts e pratique alterando o exemplo.

Exemplo

Fonte personalizada local

O navegador usa a fonte registrada pelo @font-face.

Código base

<div class="exemplo">
  <div class="amostra-fonte"><h3>Fonte personalizada local</h3><p>O navegador usa a fonte registrada pelo @font-face.</p></div>
</div>

<style>
/* GUIA DA ATIVIDADE: Registre uma fonte com @font-face e aplique a família personalizada com uma fonte de fallback. Comece pela declaração font-family indicada abaixo. Salve e compare o antes e o depois. */
@font-face {
  font-family: "FonteCurso"; /* altere esta declaração e observe o efeito */
  src: local("Georgia");
  font-style: normal;
  font-weight: 400;
}
.amostra-fonte { padding: 20px; border-left: 6px solid #b56576; background: #f7f2fa; }
.amostra-fonte h3 { font-family: "FonteCurso", Georgia, serif; font-size: 30px; margin: 0 0 8px; }
</style>

Atividade prática

Registre uma fonte com @font-face e aplique a família personalizada com uma fonte de fallback.

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
