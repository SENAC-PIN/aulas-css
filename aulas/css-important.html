CSS !important

Aprenda o papel de CSS !important e pratique alterando o exemplo.

Exemplo

A regra !important no CSS sobrescreve o estilo inline!

Código base

<div class="exemplo">
  <div class="caixa-importante estilo-inline" style="color: blue;">
A regra !important no CSS sobrescreve o estilo inline!
</div>
</div>

<style>
.caixa-importante {
  padding: 20px;
  background: #f1faee;
  border-radius: 6px;
  font-weight: bold;
  color: #e63946 !important;
}
</style>
