CSS Backgrounds

Aprenda o papel de CSS Backgrounds e pratique alterando o exemplo.

Exemplo

Cartão com fundo gradiente moderno

Código base

<div class="exemplo">
  <div class="cartao-fundo">Cartão com fundo gradiente moderno</div>
</div>

<style>
/* GUIA DA ATIVIDADE: Mude as cores, a direcao do gradiente ou use uma imagem de fundo. Comece pela declaração background-image indicada abaixo. Salve e compare o antes e o depois. */
.cartao-fundo {
  padding: 40px;
  color: white;
  font-weight: bold;
  text-align: center;
  background-color: #355070;
  background-image: linear-gradient(135deg, #355070 0%, #b56576 100%); /* altere esta declaração e observe o efeito */
  border-radius: 8px;
}
</style>

Atividade prática

Mude as cores, a direcao do gradiente ou use uma imagem de fundo.

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
