<style>
/* GUIA DA ATIVIDADE: Implemente um template mobile-first com áreas nomeadas que mudem de posição em telas maiores. Comece pela declaração grid-template-areas indicada abaixo. Salve e compare o antes e o depois. */
.template-rwd { display: grid; /* altere esta declaração e observe o efeito */ grid-template-areas: "topo" "lateral" "conteudo" "rodape"; gap: 10px; }
.template-rwd > * { padding: 20px; background: #eaf4fb; border: 1px solid #355070; }
.template-rwd header { grid-area: topo; }
.template-rwd aside { grid-area: lateral; }
.template-rwd .conteudo-template { grid-area: conteudo; }
.template-rwd footer { grid-area: rodape; }
@media (min-width: 720px) {
  .template-rwd { grid-template-columns: 220px 1fr; grid-template-areas: "topo topo" "lateral conteudo" "rodape rodape"; }
}
</style>

Atividade prática

Implemente um template mobile-first com áreas nomeadas que mudem de posição em telas maiores.

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
