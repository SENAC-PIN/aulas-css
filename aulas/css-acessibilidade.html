Código base

<div class="exemplo">
  <button class="botao-acessivel">Enviar Formulário</button>
  <span class="sr-only">(abre em nova janela)</span>
</div>

<style>
.botao-acessivel {
  padding: 12px 24px;
  font-size: 16px;
  font-weight: bold;
  background: #1a5fb4;
  color: #ffffff;
  border: none;
  border-radius: 4px;
  cursor: pointer;
}
.botao-acessivel:focus-visible {
  outline: 4px solid #f5c211;
  outline-offset: 2px;
}
.sr-only {
  position: absolute;
  width: 1px;
  height: 1px;
  padding: 0;
  margin: -1px;
  overflow: hidden;
  clip: rect(0, 0, 0, 0);
  border: 0;
}
</style>
