CSS @property

Aprenda o papel de CSS @property e pratique alterando o exemplo.

Exemplo

Propriedade customizada tipada com transição de cor

Código base

<div class="exemplo">
  <div class="caixa-property">Propriedade customizada tipada com transição de cor</div>
</div>

<style>
@property --cor-gradiente {
  syntax: '<color>';
  inherits: false;
  initial-value: #355070;
}
.caixa-property {
  padding: 30px;
  background: var(--cor-gradiente);
  color: white;
  border-radius: 8px;
  text-align: center;
  transition: --cor-gradiente 0.5s ease;
  font-weight: bold;
  cursor: pointer;
}
.caixa-property:hover {
  --cor-gradiente: #b56576;
}
</style>

Atividade prática

Registre uma custom property com @property e anime seu valor em um estado :hover.

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
