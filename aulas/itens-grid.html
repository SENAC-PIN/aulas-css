Grid Items

Crie layouts em linhas e colunas com controle preciso.

Exemplo

Destaque
2
3
4

Código base

<div class="exemplo">
  <div class="grid-itens"><div class="item-destaque">Destaque</div><div>2</div><div>3</div><div class="item-alto">4</div><div>5</div></div>
</div>

<style>
/* GUIA DA ATIVIDADE: Posicione itens específicos com grid-column e grid-row. Comece pela declaração grid-column indicada abaixo. Salve e compare o antes e o depois. */
.grid-itens { display: grid; grid-template-columns: repeat(3, 1fr); grid-auto-rows: 70px; gap: 10px; }
.grid-itens > div { display: grid; place-items: center; background: #eaf4fb; border: 1px solid #355070; }
.item-destaque { grid-column: 1 / span 2; /* altere esta declaração e observe o efeito */ }
.item-alto { grid-row: span 2; }
</style>
