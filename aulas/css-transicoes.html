CSS Transitions

Adicione movimento e mudancas visuais controladas com CSS.

Exemplo

Código base

<div class="exemplo">
  <button class="botao-transicao">Botão Suave</button>
</div>

<style>
.botao-transicao {
  padding: 12px 24px;
  background: #355070;
  color: white;
  border: none;
  border-radius: 4px;
  font-weight: bold;
  cursor: pointer;
  transition: background-color 0.4s ease, transform 0.2s ease;
}
.botao-transicao:hover {
  background: #b56576;
  transform: translateY(-3px);
}
</style>

Atividade prática

Altere o exemplo para testar outro valor ligado a CSS Transitions.

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
