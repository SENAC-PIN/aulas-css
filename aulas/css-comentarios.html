CSS Comments

Aprenda o papel de CSS Comments e pratique alterando o exemplo.

Exemplo

Passe o mouse para ver a cor mudar

Código base

<div class="exemplo">
  <div class="caixa-comentario">Passe o mouse para ver a cor mudar</div>
</div>

<style>
/* Define o design base da caixa de exemplo */
.caixa-comentario {
  padding: 20px;
  background-color: #457b9d;
  color: white;
  border-radius: 6px;
  transition: background-color 0.3s;
  text-align: center;
}
/* Altera a cor de fundo durante o hover do usuário */
.caixa-comentario:hover {
  background-color: #1d3557;
}
</style>
