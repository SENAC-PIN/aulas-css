CSS Pseudo-classes

Treine formas de escolher elementos específicos da página para aplicar estilos.

Exemplo

Código base

<div class="exemplo">
  <button class="botao-acao">Passe o mouse</button>
</div>

<style>
/* GUIA DA ATIVIDADE: Adicione estilos para :first-child ou :nth-child. Localize no exemplo a parte relacionada ao tema e altere um valor por vez. Salve e compare o antes e o depois. */
.botao-acao { padding: 12px 18px; }
.botao-acao:hover { background: #355070; color: white; }
.botao-acao:focus { outline: 3px solid #b56576; }
</style>

Atividade prática
