CSS @supports

Aprenda o papel de CSS @supports e pratique alterando o exemplo.

Exemplo

Esta caixa foi estilizada condicionalmente usando @supports.

Código base

<div class="exemplo">
  <div class="caixa-suporte">
<span class="mensagem-suporte">Esta caixa foi estilizada condicionalmente usando @supports.</span>
</div>
</div>

<style>
/* GUIA DA ATIVIDADE: Crie uma regra alternativa para quando uma propriedade for suportada. Comece pela declaração display indicada abaixo. Salve e compare o antes e o depois. */
.caixa-suporte {
  padding: 20px;
  background: #eaf4fb;
  border: 1px solid #355070;
}
@supports (display: grid) {
  .caixa-suporte {
    display: grid;
    place-items: center;
    border: 3px double #2a9d8f;
  }
}
</style>

Atividade prática

Crie uma regra alternativa para quando uma propriedade for suportada.

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
