CSS Units

Aprenda o papel de CSS Units e pratique alterando o exemplo.

Exemplo

Título em Rem (2rem)

Texto relativo ao tamanho herdado (1.1em).

Código base

<div class="exemplo">
  <div class="caixa-unidades">
    <h3 class="titulo-rem">Título em Rem (2rem)</h3>
    <p class="texto-em">Texto relativo ao tamanho herdado (1.1em).</p>
  </div>
</div>

<style>
.caixa-unidades {
  font-size: 16px;
  padding: 20px;
  background: #f8f9fa;
  border: 1px solid #ddd;
  border-radius: 6px;
}
.titulo-rem {
  font-size: 2rem;
  margin-bottom: 10px;
  color: #355070;
}
.texto-em {
  font-size: 1.1em;
  color: #666;
}
</style>
