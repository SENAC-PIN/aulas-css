CSS Box Sizing

Aprenda o papel de CSS Box Sizing e pratique alterando o exemplo.

Exemplo

Content-box
Border-box

Código base

<div class="exemplo">
  <div class="conteiner-caixas">
<div class="caixa-content">Content-box</div>
<div class="caixa-border">Border-box</div>
</div>
</div>

<style>
/* GUIA DA ATIVIDADE: Compare border-box com content-box e calcule o tamanho final. Comece pela declaração box-sizing indicada abaixo. Salve e compare o antes e o depois. */
.conteiner-caixas {
  display: flex;
  gap: 20px;
}
.caixa-content, .caixa-border {
  width: 150px;
  padding: 20px;
  border: 10px solid #1d3557;
  background: #e2eafc;
  text-align: center;
}
.caixa-content {
  box-sizing: content-box; /* altere esta declaração e observe o efeito */
}
.caixa-border {
  box-sizing: border-box;
}
</style>

Atividade prática

Compare border-box com content-box e calcule o tamanho final.

  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
