Transformações 2D CSS

Adicione movimento e mudancas visuais controladas com CSS.

Exemplo

Aprendendo CSS na pratica

Código base

<div class="exemplo">
  <div class="cartao-exemplo">Aprendendo CSS na pratica</div>
</div>

<style>
.cartao-exemplo {
  padding: 24px;
  background: #eaf4fb;
  transform: rotate(-3deg) scale(1.05);
}
</style>

Atividade prática

Teste translate(), rotate(), scale() e skew().

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
