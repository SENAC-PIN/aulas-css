CSS 2D Transforms

Adicione movimento e mudancas visuais controladas com CSS.

Exemplo

Passe o mouse para rotacionar e ampliar em 2D

Código base

<div class="exemplo">
  <div class="bloco-2d">Passe o mouse para rotacionar e ampliar em 2D</div>
</div>

<style>
.bloco-2d {
  padding: 24px;
  background: #eaf4fb;
  border: 2px solid #355070;
  text-align: center;
  font-weight: bold;
  transition: transform 0.4s ease;
}
.bloco-2d:hover {
  transform: rotate(5deg) scale(1.1);
}
</style>

Atividade prática

Teste translate(), rotate(), scale() e skew().

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
