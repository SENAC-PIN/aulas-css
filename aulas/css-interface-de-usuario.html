Interface de usuário CSS

Aprenda o papel de Interface de usuário CSS e pratique alterando o exemplo.

Exemplo

Este texto não pode ser selecionado.

Código base

<div class="exemplo">
  <div class="bloco-ui">
    <p class="nao-selecionavel">Este texto não pode ser selecionado.</p>
    <textarea class="area-redimensionavel" placeholder="Redimensione esta caixa apenas na horizontal..."></textarea>
  </div>
</div>

<style>
.bloco-ui {
  padding: 15px;
  background: #fafafa;
  border: 1px solid #ddd;
  border-radius: 6px;
}
.nao-selecionavel {
  user-select: none;
  cursor: not-allowed;
  color: #888;
}
.area-redimensionavel {
  width: 100%;
  height: 80px;
  resize: horizontal;
  padding: 8px;
  border: 1px solid #ccc;
  border-radius: 4px;
}
</style>

Atividade prática

Altere o exemplo para testar outro valor ligado a Interface de usuário CSS.

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
