Aprenda o papel de CSS Text e pratique alterando o exemplo.

Exemplo

Texto alinhado, decorado e transformado com CSS.

Código base

<div class="exemplo">
  <p class="texto">Texto alinhado, decorado e transformado com CSS.</p>
</div>

<style>
/* GUIA DA ATIVIDADE: Teste font-size, line-height, text-align e text-shadow. Comece pela declaração text-align indicada abaixo. Salve e compare o antes e o depois. */
.texto {
  font-family: Georgia, serif;
  font-size: 24px;
  text-align: center; /* altere esta declaração e observe o efeito */
  text-transform: uppercase;
  text-shadow: 2px 2px 0 #cdd7e2;
}
</style>

Atividade prática

Teste font-size, line-height, text-align e text-shadow.

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
