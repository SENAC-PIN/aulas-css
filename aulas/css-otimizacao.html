CSS Optimization

Aprenda o papel de CSS Optimization e pratique alterando o exemplo.

Exemplo

Código base

<div class="exemplo">
  <div class="container-otimizador">
    <button class="btn btn-principal">Principal</button>
    <button class="btn btn-secundario">Secundário</button>
  </div>
</div>

<style>
.btn {
  padding: 10px 20px;
  border: none;
  border-radius: 4px;
  font-weight: bold;
  cursor: pointer;
}
.btn-principal {
  background: #355070;
  color: white;
}
.btn-secundario {
  background: #eaf4fb;
  color: #355070;
  border: 1px solid #355070;
}
</style>

Atividade prática

Altere o exemplo para testar outro valor ligado a CSS Optimization.

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
