Gradiente Linear de Exemplo

Código base

<div class="exemplo">

</div>

<style>
/* GUIA DA ATIVIDADE: Mude as cores, a direcao do gradiente ou use uma imagem de fundo. Comece pela declaração background indicada abaixo. Salve e compare o antes e o depois. */
.caixa-gradiente {
  padding: 40px;
  color: white;
  font-weight: bold;
  text-align: center;
  background: radial-gradient(circle, #f8ad9d 0%, #f4978e 50%, #f08080 100%); /* altere esta declaração e observe o efeito */
  border-radius: 10px;
}
</style>
