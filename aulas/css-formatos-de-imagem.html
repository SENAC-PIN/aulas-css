CSS Image Shapes

Controle apresentação, corte, filtro e posicionamento de imagens.

Exemplo

Código base

<div class="exemplo">
  <div class="formas-imagem"><img class="forma-circulo" src="../assets/imagem-exemplo.svg" alt="Imagem recortada em círculo"><img class="forma-poligono" src="../assets/imagem-exemplo.svg" alt="Imagem recortada em polígono"></div>
</div>

<style>
/* GUIA DA ATIVIDADE: Crie dois recortes diferentes com clip-path, como circle() e polygon(). Comece pela declaração clip-path indicada abaixo. Salve e compare o antes e o depois. */
.formas-imagem { display: flex; flex-wrap: wrap; gap: 18px; }
.formas-imagem img { width: 180px; height: 150px; object-fit: cover; }
.forma-circulo { clip-path: circle(45% at 50% 50%); /* altere esta declaração e observe o efeito */ }
.forma-poligono { clip-path: polygon(50% 0, 100% 38%, 82% 100%, 18% 100%, 0 38%); }
</style>
