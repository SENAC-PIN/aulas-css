CSS SASS

Aprenda o papel de CSS SASS e pratique alterando o exemplo.

Exemplo

SCSS

$cor: #355070;
                .cartao-sass {
                  background: $cor;
                  &:hover { opacity: .8; }
                }

CSS compilado

.cartao-sass { background: #355070; }
                .cartao-sass:hover { opacity: .8; }

Código base

// styles.scss
$cor: #355070;
.cartao-sass {
  background: $cor;
  &:hover { opacity: .8; }
}

/* CSS compilado */
.cartao-sass { background: #355070; }
.cartao-sass:hover { opacity: .8; }

Atividade prática

Escreva SCSS com uma variável e nesting e apresente o CSS equivalente após a compilação.

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
