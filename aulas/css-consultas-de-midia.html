CSS Media Queries

Adapte a página para telas pequenas, medias e grandes.

Exemplo

Tela Grande (Desktop) Tela Pequena (Mobile)

Código base

<div class="exemplo">
  <div class="caixa-media">
<span class="status-desktop">Tela Grande (Desktop)</span>
<span class="status-mobile">Tela Pequena (Mobile)</span>
</div>
</div>

<style>
.caixa-media {
  padding: 24px;
  background: #a8dadc;
  border-radius: 8px;
  text-align: center;
  font-weight: bold;
}
.caixa-media .status-mobile { display: none; }
.caixa-media .status-desktop { display: inline; color: #1d3557; }
@media (max-width: 600px) {
  .caixa-media {
    background: #e63946;
  }
  .caixa-media .status-desktop { display: none; }
  .caixa-media .status-mobile { display: inline; color: white; }
}
</style>

Atividade prática

Altere o breakpoint para 800px e teste a largura da tela.

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
