Aprenda o papel de Preenchimento CSS e pratique alterando o exemplo.

Exemplo

Espaçamento interno (Padding) diferente em cada lado

Código base

<div class="exemplo">
  <div class="caixa-padding">Espaçamento interno (Padding) diferente em cada lado</div>
</div>

<style>
.caixa-padding {
  padding: 10px 40px 20px 8px;
  background: #eaf4fb;
  border: 3px solid #355070;
  font-weight: bold;
}
</style>

Atividade prática

Compare padding igual em todos os lados com padding individual.

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
