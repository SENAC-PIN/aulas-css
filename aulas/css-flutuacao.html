CSS Float

Aprenda o papel de CSS Float e pratique alterando o exemplo.

Exemplo

Texto contornando uma caixa com float.

Código base

<div class="exemplo">
  <div class="exemplo-flutuacao"><div class="miniatura"></div><p>Texto contornando uma caixa com float.</p></div>
</div>

<style>
.miniatura { float: left; width: 120px; height: 90px; margin: 0 16px 8px 0; background: #355070; }
.exemplo-flutuacao::after { content: ""; display: block; clear: both; }
</style>

Atividade prática
