Herança CSS

Aprenda o papel de Herança CSS e pratique alterando o exemplo.

Exemplo

Este parágrafo herda a cor vermelha e a fonte do elemento pai.

Esta tag span também herda as propriedades de texto.

Código base

<div class="exemplo">
  <div class="pai-heranca">
    <p>Este parágrafo herda a cor vermelha e a fonte do elemento pai.</p>
    <span class="filho-sem-borda">Esta tag span também herda as propriedades de texto.</span>
  </div>
</div>

<style>
.pai-heranca {
  color: #e63946;
  font-family: 'Courier New', Courier, monospace;
  font-weight: bold;
  border: 2px solid #1d3557;
  padding: 20px;
  border-radius: 6px;
}
.filho-sem-borda {
  display: block;
  margin-top: 10px;
  font-size: 14px;
}
</style>

Atividade prática

Altere o exemplo para testar outro valor ligado a Herança CSS.

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
