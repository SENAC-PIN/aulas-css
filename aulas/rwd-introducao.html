RWD Intro

Adapte a página para telas pequenas, medias e grandes.

Exemplo

Redimensione a janela para observar este bloco se ajustar à largura disponível.

Código base

<div class="exemplo">
  <div class="bloco-flexivel">
    Redimensione a janela para observar este bloco se ajustar à largura disponível.
  </div>
</div>

<style>
.bloco-flexivel {
  width: 100%;
  max-width: 500px;
  padding: 24px;
  box-sizing: border-box;
  background: #eaf4fb;
  border: 2px dashed #355070;
  text-align: center;
  border-radius: 8px;
  margin: 0 auto;
}
</style>

Atividade prática

Altere o breakpoint para 800px e teste a largura da tela.

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
