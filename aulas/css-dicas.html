CSS Tooltips

Aprenda o papel de CSS Tooltips e pratique alterando o exemplo.

Exemplo

Passe o mouse aqui para ver o balão.

Código base

<div class="exemplo">
  <div class="conteiner-dica">
Passe o mouse <span class="texto-dica" data-dica="Esta é uma dica flutuante!">aqui</span> para ver o balão.
</div>
</div>

<style>
/* GUIA DA ATIVIDADE: Altere o exemplo para testar outro valor ligado a CSS Tooltips. Comece pela declaração opacity indicada abaixo. Salve e compare o antes e o depois. */
.conteiner-dica {
  padding: 20px;
  text-align: center;
}
.texto-dica {
  color: #e63946;
  font-weight: bold;
  text-decoration: underline;
  position: relative;
  cursor: help;
}
.texto-dica::after {
  content: attr(data-dica);
  position: absolute;
  bottom: 125%;
  left: 50%;
  transform: translateX(-50%);
  background: #1d3557;
  color: white;
  padding: 6px 10px;
  border-radius: 4px;
  font-size: 14px;
  white-space: nowrap;
  opacity: 0; /* altere esta declaração e observe o efeito */
  visibility: hidden;
  transition: opacity 0.2s ease;
}
.texto-dica:hover::after {
  opacity: 1;
  visibility: visible;
}
</style>
