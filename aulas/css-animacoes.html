CSS Animations

Adicione movimento e mudancas visuais controladas com CSS.

Exemplo

Alerta Ativo

Código base

<div class="exemplo">
  <div class="alerta-pulsante">Alerta Ativo</div>
</div>

<style>
/* GUIA DA ATIVIDADE: Altere o exemplo para testar outro valor ligado a CSS Animations. Comece pela declaração animation indicada abaixo. Salve e compare o antes e o depois. */
.alerta-pulsante {
  width: 120px;
  padding: 12px;
  background: #e63946;
  color: white;
  text-align: center;
  font-weight: bold;
  border-radius: 50px;
  animation: pulsar 2s infinite ease-in-out; /* altere esta declaração e observe o efeito */
}
@keyframes pulsar {
  0% { transform: scale(1); box-shadow: 0 0 0 0 rgba(230, 57, 70, 0.7); }
  70% { transform: scale(1.05); box-shadow: 0 0 0 15px rgba(230, 57, 70, 0); }
  100% { transform: scale(1); box-shadow: 0 0 0 0 rgba(230, 57, 70, 0); }
}
</style>
