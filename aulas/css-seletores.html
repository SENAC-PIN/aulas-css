Seletores CSS

Treine formas de escolher elementos específicos da página para aplicar estilos.

Título com ID

Código base

<div class="exemplo">
  <div class="grupo-seletores">
    <h3 id="titulo-destaque">Título com ID</h3>
    <p class="paragrafo-normal">Parágrafo com classe específica.</p>
    <p>Parágrafo simples sem classe.</p>
  </div>
</div>

<style>
.grupo-seletores p {
  margin: 4px 0;
}
.paragrafo-normal {
  color: #355070;
  font-style: italic;
}
#titulo-destaque {
  color: #b56576;
  text-transform: uppercase;
}
</style>
