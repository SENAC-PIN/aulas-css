Múltiplas colunas CSS

Aprenda o papel de Múltiplas colunas CSS e pratique alterando o exemplo.

Exemplo

CSS permite dividir textos longos em várias colunas organizadas de forma automática, melhorando a leitura e a experiência visual do estudante em layouts responsivos.

Código base

<div class="exemplo">
  <div class="texto-multi-colunas">
    CSS permite dividir textos longos em várias colunas organizadas de forma automática, melhorando a leitura e a experiência visual do estudante em layouts responsivos.
  </div>
</div>

<style>
.texto-multi-colunas {
  column-count: 3;
  column-gap: 20px;
  column-rule: 2px solid #355070;
  padding: 15px;
  background: #f8f9fa;
  border-radius: 6px;
  font-size: 14px;
  line-height: 1.6;
}
</style>

Altere o exemplo para testar outro valor ligado a Múltiplas colunas CSS.

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
