CSS Borders

Aprenda o papel de CSS Borders e pratique alterando o exemplo.

Exemplo

Elemento com bordas customizadas

Código base

<div class="exemplo">
  <div class="cartao-borda">Elemento com bordas customizadas</div>
</div>

<style>
/* GUIA DA ATIVIDADE: Teste solid, dashed, dotted, border-radius e outline-offset. Comece pela declaração border-style indicada abaixo. Salve e compare o antes e o depois. */
.cartao-borda {
  padding: 24px;
  border-style: solid dashed double; /* altere esta declaração e observe o efeito */
  border-width: 4px 2px 8px;
  border-color: #e63946 #457b9d #1d3557;
  border-radius: 12px;
  background: #f1faee;
  text-align: center;
  font-weight: bold;
}
</style>
