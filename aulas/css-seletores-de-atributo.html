Seletores de atributo CSS

Treine formas de escolher elementos especificos da pagina para aplicar estilos.

Exemplo

Código base

<div class="exemplo">
  <form class="formulario"><label>Nome<input></label><label>Email<input type="email"></label><button>Enviar</button></form>
</div>

<style>
.formulario { display: grid; gap: 10px; max-width: 280px; }
.formulario input { padding: 10px; border: 1px solid #9fb0c1; }
.formulario input[type="email"] { border-color: #b56576; }
.formulario button { padding: 10px; background: #355070; color: white; border: 0; }
</style>

Atividade prática

Adicione estilo para input:focus e input[type='email'].

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
