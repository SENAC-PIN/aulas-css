CSS 3D Transforms

Adicione movimento e mudancas visuais controladas com CSS.

Exemplo

Efeito Rotativo 3D no Hover

Código base

<div class="exemplo">
  <div class="perspectiva-3d">
<div class="bloco-3d">Efeito Rotativo 3D no Hover</div>
</div>
</div>

<style>
.perspectiva-3d {
  perspective: 800px;
  padding: 20px;
}
.bloco-3d {
  padding: 30px;
  background: #355070;
  color: white;
  text-align: center;
  font-weight: bold;
  border-radius: 8px;
  transition: transform 0.5s;
  transform-style: preserve-3d;
}
.perspectiva-3d:hover .bloco-3d {
  transform: rotateY(30deg) rotateX(15deg);
}
</style>

Atividade prática

Altere perspective e rotateY.

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
