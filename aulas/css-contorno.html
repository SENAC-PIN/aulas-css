Exemplo

Espaço entre a borda e o contorno

Código base

<div class="exemplo">
  <div class="bloco-contorno">Espaço entre a borda e o contorno</div>
</div>

<style>
/* GUIA DA ATIVIDADE: Teste solid, dashed, dotted, border-radius e outline-offset. Comece pela declaração outline indicada abaixo. Salve e compare o antes e o depois. */
.bloco-contorno {
  padding: 20px;
  background: #f1faee;
  border: 3px solid #1d3557;
  outline: 3px solid #e63946; /* altere esta declaração e observe o efeito */
  outline-offset: 8px;
  margin: 15px;
  text-align: center;
}
</style>
