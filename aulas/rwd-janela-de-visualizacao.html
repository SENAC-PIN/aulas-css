RWD Viewport

Adapte a página para telas pequenas, medias e grandes.

Exemplo

Tamanho de texto responsivo proporcional ao Viewport (vw).

Código base

<div class="exemplo">
  <div class="conteiner-viewport">
    <p class="texto-responsivo-vw">Tamanho de texto responsivo proporcional ao Viewport (vw).</p>
  </div>
</div>

<style>
.conteiner-viewport {
  padding: 20px;
  background: #f8f9fa;
  border-radius: 6px;
  text-align: center;
}
.texto-responsivo-vw {
  font-size: 3.5vw;
  color: #355070;
  font-weight: bold;
}
</style>

Atividade prática

Altere o breakpoint para 800px e teste a largura da tela.

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
