Exemplo

Caixa de Teste

Redimensione a janela do navegador para ver o limite da largura máxima.

Código base

<div class="exemplo">
  <div class="bloco-dimensoes">
    <h3>Caixa de Teste</h3>
    <p>Redimensione a janela do navegador para ver o limite da largura máxima.</p>
  </div>
</div>

<style>
.bloco-dimensoes {
  width: 80%;
  max-width: 450px;
  height: auto;
  min-height: 150px;
  padding: 20px;
  background: #e1f5fe;
  border: 2px solid #0288d1;
  border-radius: 8px;
}
</style>

Atividade prática

Aumente e reduza a janela para observar width e max-width.

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
