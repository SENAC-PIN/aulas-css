CSS Lists

Aprenda o papel de CSS Lists e pratique alterando o exemplo.

Exemplo

  1. Planejar
  2. Codar
  3. Revisar

Código base

<div class="exemplo">
  <ol class="etapas"><li>Planejar</li><li>Codar</li><li>Revisar</li></ol>
</div>

<style>
.etapas { counter-reset: etapa; list-style: none; padding-left: 0; }
.etapas li::before {
  counter-increment: etapa;
  content: "Etapa " counter(etapa) ": ";
  font-weight: bold;
  color: #355070;
}
</style>
