CSS Shadows

Aprenda o papel de CSS Shadows e pratique alterando o exemplo.

Exemplo

Texto e Caixa Sombreados

Código base

<div class="exemplo">
  <div class="cartao-sombras">
    <h3 class="texto-sombra">Texto e Caixa Sombreados</h3>
  </div>
</div>

<style>
.cartao-sombras {
  padding: 24px;
  background: white;
  border-radius: 8px;
  box-shadow: 0 10px 20px rgba(0,0,0,0.15), 0 3px 6px rgba(0,0,0,0.1);
  text-align: center;
}
.texto-sombra {
  color: #355070;
  text-shadow: 2px 2px 4px rgba(53, 80, 112, 0.3);
}
</style>

  3. Anote qual propriedade mudou o resultado visual.
