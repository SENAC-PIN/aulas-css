Sprites de imagem CSS

Controle apresentação, corte, filtro e posicionamento de imagens.

Exemplo

Código base

<div class="exemplo">
  <img class="foto" alt="Bloco colorido" src="data:image/svg+xml,%3Csvg%20xmlns=%22http://www.w3.org/2000/svg%22%20viewBox=%220%200%20320%20180%22%3E%3Crect%20width=%22320%22%20height=%22180%22%20fill=%22%23355070%22/%3E%3Ccircle%20cx=%22230%22%20cy=%2280%22%20r=%2260%22%20fill=%22%23b56576%22/%3E%3C/svg%3E">
</div>

<style>
.foto {
  width: 260px;
  height: 150px;
  object-fit: cover;
  object-position: center;
  border-radius: 14px;
  filter: saturate(1.3);
  display: block;
  margin: auto;
}
</style>

Atividade prática

Teste object-fit, object-position, filter, border-radius e clip-path.

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
