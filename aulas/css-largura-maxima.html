CSS Max-width

Aprenda o papel de CSS Max-width e pratique alterando o exemplo.

Exemplo

Largura máxima de 400px (Responsivo)

Código base

<div class="exemplo">
  <div class="conteiner-larguras">
    <div class="caixa-maxima">Largura máxima de 400px (Responsivo)</div>
  </div>
</div>

<style>
.caixa-maxima {
  width: 100%;
  max-width: 400px;
  padding: 24px;
  background: #eaf4fb;
  border: 2px solid #355070;
  border-radius: 8px;
  box-sizing: border-box;
}
</style>

Atividade prática

Aumente e reduza a janela para observar width e max-width.

  1. Abra esta página no navegador.
  2. Edite o CSS no arquivo ou no DevTools.
  3. Anote qual propriedade mudou o resultado visual.
